Opções disponiveis

-b : Retorna informações da base do registro.br

-a : retorna informações configuradas para o dominio

di -b xxxxxxxxx-xx || di -d domain.com.br

di -a domain.com

root@urmachine:~# di -a eroscarvalho.com.br

Entrada SOA :

ns1.dns.com.br. dnsadmin.br1.dns.com.br.

Servidores DNS :

ns1.dns.com.br.

ns2.dns.com.br.

IP para dominio.com.br :

xxx.xxx.xxx.xxx

Entradas MX :

0 dominio.com.br.

Entradas SPF/TXT :

"v=spf1 a mx include:websitewelcome.com ~all"
